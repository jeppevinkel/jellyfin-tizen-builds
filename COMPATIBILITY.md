## Legend

- **Jellyfin-web Version**: The version of Jellyfin-web supported
  - `>= 10.11.z` - Supports version 10.11 and newer
  - `<= 10.9.z` - Only supports up to version 10.9
  - Specific version (e.g., `10.8.13`) - Only supports that exact version
- **TrueHD Support**: Whether the TV can decode TrueHD audio
  - `true` - Confirmed support
  - `false` - Confirmed no support
  - `unknown` - Not tested/verified
- **SmartHub Support**: Whether SmartHub features are available
  - `true` - Confirmed support
  - `false` - Confirmed no support
  - `unknown` - Not tested/verified
